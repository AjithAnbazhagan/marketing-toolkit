# Marketing Toolkit

A free AI-powered marketing toolkit built by Ajith Anbazhagan.

## Tools

- **Repurpose Agent** — Turn any blog into LinkedIn and Meta content instantly
- **Marketing Decision Journal** — Log campaign decisions and auto-generate stakeholder reports
- **Ad Copy Studio** — Write and preview ads for LinkedIn, Meta and Google with live character counts
- **UTM Builder** — Build clean, consistent UTM tracking links with naming convention rules
